# Recall Format Reference
*Permanent template for Echo Memory Recall output.*

## Single Match Output
```
Yes, I remember! On [date], we [summary of what happened].
[Relevant excerpt from diary entry].
[Connection to current conversation].
```

## Multiple Matches Output
```
I found a few relevant sessions:

1. [Date]: [Summary] — [Key detail]
2. [Date]: [Summary] — [Key detail]

[Pattern observation if applicable].
```

## No Match Output
```
I don't have a record of [topic] in my diary. Can you tell me more about what you're remembering?
```

## Uncertain Match Output
```
I found something that might be related: [date] — [summary]. Is that what you're thinking of?
```
